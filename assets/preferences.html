The Device Setup Dialog

OS Type
Choose the operating system your Kodi runs on. The default is Android. Changing the operating system also changes the file path to Kodi's data folder to the default for the chosen OS.

Description
Enter a descriptive title for your device.

Address (Android only)
Enter the device IP address. A device name is also allowable if you have set it up with your router.

Package name (Android only), default: org.xbmc.kodi.
Select the package name by choosing your media center software from the Media Center presets, or enter package name directly. If you are running Kodi leave this setting unchanged.

File Path
This is the location of Kodi's data on your device. This location varies depending on the OS Kodi is running on and can be entered in different ways:

Change the Kodi OS to select a default value per OS

Select a Media Center preset (Android only)
Available presets:
Kodi
SPMC
Other

File Path button
Browse the adbLink filesystem to locate your file path, useful if Kodi's data is in a non-standard location (e.g. a network share).

Manually enter

Pull Folder (Android only), default: User's home directory.
Choose the folder where pulled Kodi files and screenshot pngs are placed.

The Check for updates at startup checkbox causes adbLink to check jocala.com for updates at startup. This is on by default.

If you have questions or comments, please email jocala@jocala.com or visit one of the forums linked below. adbLink produces a log file, adbfire.log, with each run. Include it in your email or post if you're having problems.
