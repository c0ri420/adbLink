adbLink Overview

adbLink Main Screen

Devices
The upper left Current Device combobox contains the names of your devices. This dropdown combobox is empty when you first run adbLink. Once you add devices, you choose a device from this menu in order to interact with it.

The buttons associated with the device dropdown menu are:

New: Display a blank device settings screen (New USB devices are also added via a double-click in the Connected Devices listbox, see below).

Edit: Edit the selected device.

Delete: Delete the selected device.

Connect: Connect the selected device. (Android only)

Disconnect: Disconnect the selected device. (Android only)

The Connect and Disconnect buttons are disabled if the selected Android device is connected via USB.

Android file management:
The Preset directories dropdown menu allows you to select a directory to push files to, pull files from or delete files.

The Ad Hoc Directory entry field allows entry of any directory you have access to on the device for push/pull/delete.

The Push button allows selection of a file on the PC to be pushed to the selected directory on the connected device.

The Pull button allows selection of a file on the device to be pulled from the selected directory on the connected device.

The Del button allows selection of a file on the device for deletion.

The Clear button clears the Ad Hoc directory selection.

Device connections:
The Connected Devices (Android) listbox shows all current adb connections.

The Status (Android) listbox shows the device status (device/emulator/offline,etc).

Double-clicking a USB device ID in the Connected Devices listbox opens a new device record with the ID filled in. This allows you to easily create a device record for USB connections

The Restart ADB (Android) button: Does refresh of connected devices

The Stop ADB button (Android) stops the ADB server.

Android only buttons:
Install apk: Installs an apk file.

Uninstall apk: Uninstalls an apk file.

Screen Shot: Takes a screen shot of whatever is on your device's TV screen.

ADB Shell: Opens a console on the connected device. Much like telnet.

Remote:WIFI/USB Remote.

Move Kodi Data: Move Kodi's data to an external partition

Global buttons:
Backup: Create a Kodi backup.

Restore: Restore a Kodi backup

Cache:Adjust Kodi's Cache settings.

Splash Screen: Push a remote control XML file to Kodi

Push Remote: Push a remote control XML file to Kodi

Console: Opens a console for the command-line. Allows entry of adb/fastboot commands for Android.

File Menu:
Edit XML files: Edit Kodi XML files

View adbLink Log: View a log of your current or previous adbLink session. Copy the log to the system clipboard for pasting into messages,email,etc.

View Kodi Log: View Kodi's log files. Copy the log to the system clipboard for pasting into messages,email,etc.

Reboot device: Reboots your Android device

Reboot recovery: Reboots your Android device to recovery

Root menu: (Android only)
FS R/O:Make the file system read-only

FS R/W:Make the file system read-write
